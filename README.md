# Wallpaper of the Day

A fresh new daily wallpaper downloaded from a famous portal (Widget).

## Features

Download the daily image from Bing and set it as your desktop background.

## Installation

1. Open DMS Settings → Plugins
2. Click "Scan for Plugins"
3. Browse for `Wallpaper of the Day (Widget)` 3rd party plugin
4. Install, enable and add the widget to the bar

## Settings

No settings required.

## Requirements

- `Wallpaper of the Day daemon plugin` `inotify-tools`
